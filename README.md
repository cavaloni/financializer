# Chad's React Boilerplate

My template for developing React apps.

## Getting started

Clone locally and have fun!

Use `npm run dev` for front-end only apps.
Use `npm run dev-build` for full-stack development.

### Tech Included
- Babel 6
- Webpack 2.2
- React
- React Router 3
- Redux
- RxJS 5
- CSS Modules
- LoDash
- ESLint (AirBnB)
